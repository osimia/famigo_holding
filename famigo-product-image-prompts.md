# FAMIGO — промты генерации фото продуктов каталога (25 позиций)

## Как пользоваться
1. Промты на английском — так генераторы (Midjourney, Flux, DALL-E, Ideogram,
   Stable Diffusion) работают заметно лучше. Копируй промт целиком, каждый
   самодостаточен.
2. Для Midjourney добавь в конец: `--ar 1:1 --style raw`.
   Для Stable Diffusion / Flux используй негативный промт из конца файла.
3. Единый стиль: кремовый фон + оливковый акцент (палитра сайта), керамическая
   посуда, мягкий дневной свет, ракурс 45°. Не меняй эту часть промта — иначе
   каталог развалится визуально.
4. Генерируй 3–4 варианта на позицию, выбирай самый «честный» — без
   пластиковой глянцевости.

⚠️ **Для B2B это временная мера.** Закупщик принимает решение по реальному
виду товара (калибр, цвет партии). Сгенерированные фото годятся для запуска
каталога, но перед серьёзными экспортными переговорами замени их на фото
реальных партий — иначе первая же выборка образцов подорвёт доверие.

## Базовая формула (уже вшита в каждый промт)

warm cream background, folded olive-green linen napkin beside, soft diffused
daylight from upper left, 45-degree angle, shallow depth of field,
photorealistic, high detail, no text, no logo, no hands, no packaging.

## Негативный промт (для Stable Diffusion / Flux)
text, watermark, logo, label, packaging, plastic bag, hands, fingers, people,
cartoon, illustration, 3d render, oversaturated, glossy plastic look, blurry,
extra objects, cluttered background

## Советы по консистентности
- Midjourney: сгенерируй первое удачное фото (например курагу) и используй его
  как style reference (`--sref <url>`) для всех остальных — каталог будет в
  одном тоне.
- Один и тот же размер посуды в кадре: если бросается разнобой — добавь
  `same bowl as reference` или регенерируй.
- Итоговый размер для карточек: 1200×1200, экспорт в WebP ≤ 150 КБ.
- Имена файлов под уже существующую схему сайта: /product-photos/kuraga.webp,
  raisin-soyagi.webp и т.д.
